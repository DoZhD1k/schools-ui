# Роуты системы "Цифровой рейтинг школ"

## Основные страницы

### 🏠 Главная страница рейтинга

- **URL**: `/ru/schools/rating`
- **Компонент**: `MainRatingPage`
- **Описание**: Дашборд с интерактивными графиками, фильтрами и рейтинговой таблицей школ

### 📚 Каталог организаций

- **URL**: `/ru/schools/organizations`
- **Компонент**: `OrganizationsPage`
- **Описание**: Пагинированная таблица всех школ с расширенными фильтрами и поиском

### 📋 Паспорт школы

- **URL**: `/ru/schools/passport/[id]`
- **Компонент**: `SchoolPassportPage`
- **Описание**: Детальная информация о конкретной школе с графиками и метриками
- **Параметры**: `id` - идентификатор школы

### 🗺️ Интерактивная карта

- **URL**: `/ru/schools/map`
- **Компонент**: `SchoolMapPage`
- **Описание**: Карта с пинами школ и боковой панелью детальной информации

## Административные страницы

### 👥 Управление пользователями

- **URL**: `/ru/admin/users`
- **Компонент**: `UsersManagementPage`
- **Описание**: Административная панель для управления пользователями системы

### ⚖️ Инструмент весов критериев

- **URL**: `/ru/admin/weights`
- **Компонент**: `WeightsToolPage`
- **Описание**: Настройка весовых коэффициентов для расчета рейтинга школ

## Навигация

Все новые страницы добавлены в главное навигационное меню (`NavPanel.tsx`):

- ⭐ **Рейтинг школ** → `/ru/schools/rating`
- 👥 **Каталог** → `/ru/schools/organizations`
- 🗺️ **Карта школ** → `/ru/schools/map`
- 🛡️ **Админка** → `/ru/admin` (только для администраторов)

## Структура файлов

```
app/[lang]/(routes)/
├── schools/
│   ├── page.tsx                    # Главная страница рейтинга
│   ├── organizations/
│   │   └── page.tsx               # Каталог организаций
│   ├── passport/
│   │   └── [id]/
│   │       └── page.tsx           # Паспорт школы
│   └── map/
│       └── page.tsx               # Интерактивная карта
└── admin/
    ├── users/
    │   └── page.tsx               # Управление пользователями
    └── weights/
        └── page.tsx               # Инструмент весов критериев
```

## Компоненты

Все компоненты находятся в папке `components/schools/`:

- `main-rating-page.tsx`
- `organizations-page.tsx`
- `school-passport-page.tsx`
- `school-map-page.tsx`
- `users-management-page.tsx`
- `weights-tool-page.tsx`
