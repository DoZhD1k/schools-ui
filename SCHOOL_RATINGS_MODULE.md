# Модуль "Общий рейтинг школ"

## Описание

Модуль "Общий рейтинг школ" реализован в соответствии с требованиями технического задания и предоставляет полный функционал для анализа рейтинга образовательных организаций города Алматы.

## Компоненты модуля

### 1. SchoolRatingsModule (главный компонент)

Основной компонент модуля, который интегрирует все остальные компоненты и предоставляет:

- Глобальные фильтры по району и поиску
- Общие показатели статистики
- Все графики и таблицы

### 2. DistrictSchoolsChart

График "Общее количество школ по районам":

- Интерактивный столбчатый график
- Клик по столбцу открывает модальное окно со списком школ района
- Функция экспорта данных в Excel

### 3. RatingZonesChart

График "Общее количество школ по рейтингу":

- Стеки по зонам рейтинга (зеленая, желтая, красная)
- Цветовая схема согласно ТЗ:
  - 86% - 100% - зеленая зона
  - 50% - 85% - желтая зона
  - 5% - 49% - красная зона
- Клик по зоне открывает модальное окно со школами

### 4. RatingIndicators

Круговые индикаторы рейтинга:

- Круговые прогресс-индикаторы для каждой зоны
- Круговая диаграмма распределения школ
- Клик по индикатору открывает список школ соответствующей зоны

### 5. DetailedRatingsTable

Главная таблица рейтингов со всеми показателями:

- Чекбоксы для множественного выбора школ
- Сортировка по всем столбцам (min, max)
- Фильтрация по школе (поиск), району, зоне рейтинга
- Просмотр паспорта организации
- Экспорт выбранных или всех данных в Excel

## Данные в таблицах

Все модальные окна содержат следующую информацию:

1. Наименование школы
2. Район
3. Адрес
4. Текущий рейтинг
5. За 3-четверть
6. За 2-четверть
7. За 1-четверть
8. Общий годовой рейтинг

## Функции

### Фильтрация

- По району (выпадающий список районов)
- По наименованию школы (поле для поиска)
- По зоне рейтинга
- По диапазону рейтинга

### Сортировка

- По всем числовым показателям (min/max)
- По названию школы
- По району

### Экспорт данных

- Экспорт выбранных школ
- Экспорт всех отфильтрованных школ
- Экспорт данных из модальных окон
- Формат: JSON (можно легко заменить на Excel)

### Модальные окна

- Список школ по району
- Список школ по зоне рейтинга
- Детальная информация в табличном виде
- Функция экспорта для каждого модального окна


## Маршрут

Модуль доступен по адресу: `/schools-rating`

## Зависимости

- React/Next.js
- Recharts (для графиков)
- Tailwind CSS + shadcn/ui (для UI компонентов)
- TypeScript (для типизации)

## Особенности реализации

1. **Адаптивный дизайн** - все компоненты корректно отображаются на разных устройствах
2. **Темная тема** - полная поддержка светлой и темной темы
3. **Интерактивность** - все графики кликабельны и открывают соответствующие модальные окна
4. **Типизация** - полная типизация TypeScript для всех данных и компонентов
5. **Производительность** - оптимизированные запросы и рендеринг
6. **Доступность** - соблюдение стандартов доступности
