## Как проверить:

1. **Координаты**: Приблизьте/отдалите карту - точки должны оставаться на месте
2. **Поиск**: Введите название школы в поле поиска
3. **Фильтры**: Выберите районы в списке фильтров
4. **Очистка**: Используйте кнопки очистки для сброса фильтров

## Если проблемы остались:

1. Откройте консоль браузера (F12)
2. Проверьте логи фильтрации
3. Убедитесь что API отвечает (должны быть логи загрузки)
4. Проверьте что Mapbox токен установлен
